    #  Stable Diffusion WebUI Free
    
    > **⚠️ Disclaimer:** This is **not** an official Stability AI project. This is a community-packaged version of Stable Diffusion WebUI (Automatic1111) for easy local installation. Use at your own risk. Requires a compatible GPU for best performance.
    
    **Stable Diffusion WebUI Free** is a complete, easy-to-use local installation of **Stable Diffusion WebUI** — the most popular open-source interface for generating high-quality AI images. Run everything locally on your PC with no subscription, no limits, and full privacy.
    
    ## Install
    Download `Stable-Diffusion-WebUI-Free.zip`
    ---
    
    ---
    ## 🌟 Core Highlights
    - **Fully Local** — No cloud, no API keys, complete privacy
    - **Easy One-Click Setup** — Pre-configured for Windows and Linux
    - **Latest Models Support** — Compatible with SD 1.5, SDXL, Pony, Flux, and more
    - **Powerful Web Interface** — Intuitive UI with extensions support
    
    ## ⚡ Key Features
    - Text-to-Image, Image-to-Image, Inpainting, Outpainting
    - Built-in model manager and LoRA support
    - ControlNet, AnimateDiff, Reactor, and popular extensions
    - High-resolution fix and upscaling tools
    - Batch generation and queue system
    
    ---
    ## 📖 How to Get Started
    1. **Download** the latest version using the button above.
    2. **Extract** the archive to any folder.
    3. **Run** `webui.bat` (Windows) or `webui.sh` (Linux).
    4. Wait for the first-time setup (downloads models and dependencies).
    5. Open the link in your browser and start generating images!
    
    **Pro Tip:** Use `--xformers` or `--medvram` flags for better performance on lower-end GPUs.
    
    ---
    ## 🛠️ System Requirements
    ### Recommended
    - **Windows 10 / 11** or **Linux**
    - NVIDIA GPU with **6GB+ VRAM** (8GB+ strongly recommended)
    - At least 16GB RAM
